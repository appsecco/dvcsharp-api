# API Usage

The *DVCSharp* application supports RESTful APIs to perform various operations such as:

* Register User
* Authentication and get access token
* Get token info
* Update user
* Import user
* Delete user
* SSO authentication to get access token
* Password reset
* List products
* Create products
* Export products
* Generic import entities
